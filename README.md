## Getting Started

### Installation

To install the project on your Pharo image you can just execute the following script:

```smalltalk
Metacello new
   baseline: 'PharoZeroMQ';
   repository: 'github://OpenSmock/PharoZeroMQ:main/src';
   load.
```
